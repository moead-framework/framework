moead_framework.core.genetic_operator.combinatorial.cross_mut.CrossoverAndMutation¶

class moead_framework.core.genetic_operator.combinatorial.cross_mut.CrossoverAndMutation(solutions, crossover_points=1, mutation_probability=None)[source]¶

Bases: moead_framework.core.genetic_operator.abstract_operator.GeneticOperator

Multi-point crossover combined with the Binary Mutation operator

Require 2 solutions, run a crossover according to the number of points wanted and try to mutate each bit of the decision_vector with the predefined probability.

  • solutions – list<list<integer>> list of solution’s representation (In algorithms, it is represented by the attribute decision_vector of the class OneDimensionSolution)

  • crossover_points – {integer} the number of points for the crossover

  • mutation_probability – {float} the probability (between 0 and 1) to mutate a bit in the decision_vector. The default value is the probability to mutate one bit of the decision_vector
